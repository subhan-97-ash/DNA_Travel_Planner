**Smart Travel Planning using AI**
An intelligent travel planning web app that generates personalized itineraries using AI, real-time weather data, and travel APIs.Built with LangGraph, RAG, Streamlit, and external APIs, this app helps users plan smarter trips with dynamic recommendations.

**Work Flow**
- User enters source, destination, start date, and trip duration.
- AI generates a multi-day itinerary.
- Weather API checks forecast for each day.
- Rainy days are adjusted using RAG-based indoor activities.
- Flights and hotels are fetched using travel APIs.

**Deployment**
- This app is deployed using Streamlit Cloud.
